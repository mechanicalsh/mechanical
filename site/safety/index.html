What it certifies

Six things the seal guarantees.

Every line maps to evidence a third party can check, not a promise you have to trust.

  • A physical veto exists

    An independent core can cut power to the actuators, proven on the bench.

  • The stop path is verified

    The halt logic is formally proven, with the proof on file.

  • The model can't reach it

    No software path lets the reasoning layer disable the core.

  • Every action is logged

    Decisions and vetoes are signed and exportable, end to end.

  • Tamper is evident

    Opening or reflashing the unit voids the seal, visibly.

  • It runs the published constitution

    The unit proves which signed ruleset it is enforcing.
